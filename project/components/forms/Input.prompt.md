Dark glass text field. Label above, accent ring on focus, red border on `error`. `as="textarea"` for long text (seeker bio).

```jsx
<Input label="Full name" placeholder="e.g. Selam Tesfaye" />
<Input label="Video pitch link" placeholder="YouTube / TikTok URL" leadingIcon={<LinkIcon/>} />
<Input as="textarea" label="About you" hint="A short pitch — 2–3 sentences." />
```
